🚀 YouTube JSONL Processing Pipeline (Google Drive + AI Powered)

Turn messy .jsonl data into clean, structured, and AI-ready content — automatically.

📌 What this project does


This project is an end-to-end automation pipeline that takes YouTube data from .jsonl files and transforms it into organized assets like videos, audio files, transcripts, and thumbnails — all in one flow.

⚙️ How it works

🔗 Google Drive Integration

Connects to your Drive folder, scans, and downloads all .jsonl files effortlessly.

🧩 Smart JSON Parsing

Extracts YouTube video IDs even from deeply nested and complex data structures.

🎬 Video Processing

Downloads each video in high-quality MP4 format using yt-dlp.

🎵 Audio Extraction

Converts videos into MP3 and WAV using FFmpeg (optimized for speech processing).

🧠 AI Transcription (Whisper)

Generates transcripts directly from audio with OpenAI Whisper.

🤖 AI Correction (GPT - Optional)

Fixes transcription errors while preserving meaning and structure.

🖼️ Thumbnail Fetching

Automatically downloads and saves video thumbnails.

📂 Clean Organization

Each video gets its own folder with all assets neatly structured.
